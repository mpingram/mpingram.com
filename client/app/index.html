Acrobat Mailmerge an Acrobat Javascript extension for variable data printing

Variable data printing, if you didn't know, is how businesses do things like printing your name and address on a letter. The data (your name and address, in this case) is transferred directly to the letter as it prints - imagine the waste if a business needed to save a thousand copies of the same letter, each with a different name and address field.

If you want to print via .pdf files, however - and that's always a good idea - that's the way you have to do it: Adobe Acrobat has no native support for mailmerging or variable data printing. The Acrobat Mailmerge script adds that support by leveraging Acrobat 10+'s form fields feature, which allow text to be edited on the fly.
