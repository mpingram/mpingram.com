Acrobat Mailmerge an Acrobat Javascript extension for variable data printing

Variable data printing, if you didn't know, is how businesses do things like print your name and address on a letter. Data (for example, your name and address) is transferred directly to the letter as it prints - imagine the waste if a business needed to save a thousand copies of the same letter, each with a different name and address printed on them.

If you want to print via .pdf files, however, that's the way you have to do it: Adobe Acrobat has no native support for mailmerging or variable data printing. The Acrobat Mailmerge script adds that support by making creative use of Acrobat's form fields, updating them with data from an excel spreadsheet and silently printing the document.
